使用说明
开发新表格目的

主要是为了支持大数据量的渲染。比如一个表格有2万条数据，旧表格可能会挂掉，新表格的渲染做了优化。

新表格相关指令
指令名 含义
g-hot-table 新表格指令，用法与旧表格相同
g-hot-column 新表格列定义指令，用法与旧表格相同
g-hot-checkbox-column 新表格复选框列定义指令，实现多选功能，用法与旧表格相同
g-hot-index-column 新表格列配置序号，用法与旧表格相同
g-hot-column-editor 指令内定义列编辑的方式，用法与旧表格相同
新表格属性（g-hot-table）
参数名 类型 是否必须 说明
source-name string Y 数据源名称，保持不变
validator-name string N 校验名称，保持不变
on-load-success function(grid - 表格对象,$event - 事件对象,source - 表格数据集,physicalSource - 表格数据集) N 表格数据加载成功事件
on-render function(grid - 表格对象,source - 表格数据集) N 表格初始化后的回调事件，NOTE：此时表格尚无数据
css-class express / function() / function(row:Row) N 新表格样式属性，用法与旧表格 g-item-class 类似
stretch String N 新增属性，表格列宽度不足表格宽度时，如何处理空白部分(none, last, all)
width, height Number/String N 表格宽度及高度
on-row-click function(grid - 表格对象,$event - 事件对象,record - 当前点击行数据对象,colIndex - 当前点击所在的列序号,rowIndex - 当前点击所在的行序号（不包含隐藏行）,field - 当前点击所在的列字段名称,physicalRow - 当前点击所在的行序号（包含隐藏行）) N 鼠标单击行事件
on-row-dbclick function($event - 事件对象,grid - 表格对象,record - 当前点击行数据对象,colIndex - 当前点击所在的列序号,rowIndex - 当前点击所在的行序号（不包含隐藏行）,field - 当前点击所在的列字段名称,physicalRow - 当前点击所在的行序号（包含隐藏行）) N 鼠标双击行事件
on-check function(grid - 表格对象,source - 表格数据集（层次数据结构）,physicalSource - 表格数据集（扁平数据结构）,checked - 勾选状态,row - 勾选行,physicalRow - 行序号,record - 行数据对象) N 切换行勾选状态事件
on-check-all function(grid - 表格对象,$event - 事件对象,source - 表格数据集（层次数据结构）,physicalSource - 表格数据集（扁平数据结构）,checked - 全选状态) N 切换全选选状态事件
col-settings-key String N 页面布局名称，保持不变
tree-view N 是否开启树装表格功能
group Boolean N 是否开启分组表格功能
drag-check Boolean N 是否开启拖动勾选，保持不变
filters N 是否启用下拉过滤功能
front-end-sort Boolean N 是否使用前端排序方式，默认为请求后端进行排序
front-end-group Boolean N 是否使用前端分组方式，默认为请求后端进行分组
group-summary-func Boolean N 分组合计方法
validate-on-change-editing-row / validate-on-change-editing-cell N 新增属性，是否在切换编辑行/单元格时执行校验
列定义属性（g-hot-column）
参数名 类型 是否必须 说明
data express N 列字段，支持 angular 过滤器表达式，保持不变
sortable void N 是否可排序，保持不变
width Number/String N 列宽度，保持不变
g-hot-column-editor N 指令内定义列编辑的方式
align string N 列对齐方式
required Boolean N 是否必填字段，保持不变
hidden Boolean N 是否隐藏列，保持不变
title String N 列标题文字
复选框指令（g-hot-checkbox-column）
参数名 类型 是否必须 说明
width Number/String N 宽度属性
列配置序号指令（g-hot-index-column）
表格对象方法（grid）
方法名 使用方式 说明
getSelectedRow grid.getSelectedRow() 用于获取选中行数据
getCheckedRows grid.getCheckedRows() 用于获取勾选数据集合
startEdit grid.startEdit(row, col) 开始编辑表格单元格
finishEdit grid.finishEdit(validateType) 结束编辑表格单元格
addRow grid.addRow(row,newRecord) 新增行
deleteRow grid.deleteRow(physicalRow) 删除行
validateCell grid.validateCell(physicalRow, field) 校验某个单元格数据
validateRow grid.validateRow(physicalRow) 校验某行数据
validateAddedAndModifiedRows grid.validateAddedAndModifiedRows() 校验所有新增行及修改行的数据
getCheckedLeafRows grid.getCheckedLeafRows() 获取分组表格的叶子节点数据集合
getCheckedGroupRows grid.getCheckedGroupRows() 获取分组表格的分组行数据集合
简单用例
  • 该控件不是公共控件需要手动引入framework/hotTable/HotTableModule模块
    • <script src="/bower_components/requirejs/require.js" data-main="/static/app.js"
        data-bootstrap-modules="[ 'framework/hotTable/HotTableModule',
                '/gschool/try/TestModule']">
</script>
  • 在html页面上使用新表格控件展示数据源信息
    • <g-hot-table ng-cloak data-source-name="employeeSource"
             on-load-success="empGridSuccess(grid, source)"
             checked-rows="empCheckedRows"
             col-settings-key="empListSettings"
             >
    <g-hot-index-column></g-hot-index-column>
    <g-hot-checkbox-column></g-hot-checkbox-column>
    <g-hot-column data="name" sortable align="center" width="120" required title="姓名">
        <g-hot-column-editor>
            <input type="text" name="name" ng-model="row.name">
            <input type="hidden" name="id" ng-model="row.id"/>
        </g-hot-column-editor>
    </g-hot-column>
    <g-hot-column data="sex | sexToCn" sortable width="100" title="性别">
        <g-hot-column-editor>
            <g-dropdown list-type="single"
                        placeholder="请选择"
                        source="sexData"
                        display-express="name"
                        value-prop="value"
                        name="sex"
                        ng-model="sex">
            </g-dropdown>
        </g-hot-column-editor>
    </g-hot-column>
    <g-hot-column data="birthDay" sortable width="200" title="出生日期">
    </g-hot-column>
    <g-hot-column data="age" sortable width="250" title="年龄">
    </g-hot-column>
</g-hot-table>
